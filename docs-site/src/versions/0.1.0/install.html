Starter commands

opengate init --template website
opengate init --template api
opengate init --template partner
opengate init --route mixed

What the starter gives you

  • JSON config with the Phase 1 security defaults.
  • Starter server code for the selected template.
  • Demo credentials and sample audit data.
  • A clear path from root compatibility package to explicit adapter packages.
